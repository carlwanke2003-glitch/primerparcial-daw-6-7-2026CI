Tecnologías

HTML

Es el esqueleto. Define dónde van la cabeza, los brazos y las piernas.

CSS

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Si el HTML es el esqueleto de una página web, el CSS es el diseño, la pintura y la estética que la hacen lucir profesional.

PHP

PHP es un lenguaje de programación de código abierto, interpretado y orientado al desarrollo web del lado del servidor.

JAVASCRIPT

JavaScript es el entorno en el que se evalúa y ejecuta el código, gestionando la memoria, el alcance (scope) y el valor de this.
